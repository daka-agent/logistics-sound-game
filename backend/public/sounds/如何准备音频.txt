# 占位音频说明

由于音频文件需要自行准备，这里提供一个快速测试方案：

## 使用任意短音频文件

可以将任意一个短音频文件（3-5秒）复制10份，分别重命名为：

1. truck-engine.mp3
2. forklift-reverse.mp3
3. scanner-beep.mp3
4. conveyor-belt.mp3
5. truck-brake.mp3
6. forklift-hydraulic.mp3
7. package-drop.mp3
8. container-door.mp3
9. pallet-collision.mp3
10. packing-tape.mp3

## 或使用在线音频URL

修改 backend/data/questions.json 中的 audio_url 字段，使用在线音频链接，例如：

```json
{
  "audio_url": "https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3"
}
```

注意：使用在线音频可能受跨域限制。
