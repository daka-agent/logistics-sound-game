# 音频素材说明

本目录用于存放游戏所需的物流声音素材（MP3格式）。

## 所需音频文件

根据 `backend/data/questions.json` 中的配置，需要以下音频文件：

### 运输车辆类
- truck-engine.mp3 - 卡车引擎启动声
- truck-brake.mp3 - 卡车气刹声
- container-door.mp3 - 集装箱门开合声

### 装卸设备类
- forklift-reverse.mp3 - 叉车倒车提示音
- forklift-hydraulic.mp3 - 叉车液压升降声
- pallet-collision.mp3 - 托盘碰撞声

### 仓储设施类
- scanner-beep.mp3 - 扫码枪滴声
- conveyor-belt.mp3 - 传送带运转声
- package-drop.mp3 - 包裹滑落声
- packing-tape.mp3 - 胶带封箱声

## 音频要求

- 格式: MP3
- 时长: 3-8秒
- 大小: ≤ 500KB/个
- 质量: 清晰可辨，噪音适中

## 获取音频资源

推荐以下免费音效网站：

1. **Freesound.org** (https://freesound.org)
   - 最大免费音效社区
   - Creative Commons许可
   - 需注册账号下载

2. **Pixabay Sound Effects** (https://pixabay.com/sound-effects/)
   - 无需注册
   - 可商用免费
   - 分类清晰

3. **Mixkit** (https://mixkit.co/free-sound-effects/)
   - 免费使用
   - 质量较高

4. **Zapsplat** (https://www.zapsplat.com)
   - 专业音效
   - 免费版有限制

## 搜索关键词

搜索时可使用以下关键词：
- truck engine

- conveyor belt
- pallet
- cargo
- warehouse

## 临时测试

在正式音频素材准备好之前，游戏可以正常运行，但音频播放会失败。

如需快速测试，可以使用任意短音频文件重命名为上述文件名。
